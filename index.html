Polymer Internationalization (i18n) Component

  • Priority Precedence :
    • Query parameter(Eg: lang=fr)
    • Browser language

  • Usage :
    • For Static content management
    • Create folder structure as locales/en-US/translation.json for english based on the language you have to create folder under the locales with a particular translation.json
    • import <i18m-component> as a child component to your customize component
    • add i18n as a parameter as <i18m-component i18-n="{{i18n}}" >
    • Then call to words as yor structure in translation.json
